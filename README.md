## 📖 快速开始

### 电脑端（Clash Verge Rev）
1. 下载安装 Clash Verge Rev
2. 打开 → 配置 → 导入 → 从URL导入
3. 粘贴订阅链接 → 保存
4. 点击启动 → 选择节点

### 安卓端（Clash Meta）
1. 下载安装 Clash Meta for Android
2. 打开 → 配置 → + → 从URL导入
3. 粘贴订阅链接 → 保存
4. 点击启动 → 允许VPN → 选择节点

### iOS端（Shadowrocket）
1. 需要美区/港区 Apple ID
2. App Store 下载 Shadowrocket
3. 右上角 + → 类型选 Subscribe
4. URL 粘贴订阅链接 → 保存
5. 选择节点 → 开启
